使用Miniconda

1.下载软件包

xxxxxxxxxx
wget https://mirrors.tuna.tsinghua.edu.cn/anaconda/miniconda/Miniconda3-latest-Linux-x86_64.sh

 

2.执行安装程序

xxxxxxxxxx
bash Miniconda3-latest-Linux-x86_64.sh

利用 Ctrl+C 跳过阅读教程，最后yes 确认安装，注意安装路径

3.检查环境变量

输入 conda 检查是否已添加环境变量，如果没有则需要进行配置环境变量

xxxxxxxxxx
vim ~/.bashrc

打开vim界面后，按 i 进入编辑模式，插入一行

xxxxxxxxxx
export PATH=/home/user/miniconda3/bin:$PATH

随后按 Esc 退出编辑模式，按 :wq 保存并退出

xxxxxxxxxx
source ~/.bashrc

进行环境变量生效

4. 更改conda默认环境位置

利用

xxxxxxxxxx
conda info

查看目前conda的package和envs路径，打开condarc文件：

xxxxxxxxxx
sudo vim ~/.condarc

插入

xxxxxxxxxx
envs_dirs:
- xxx/xxx/xxx(新的虚拟环境路径)
pkgs_dirs:
- xxx/xxx/xxx(新的包缓存路径)

后保存退出。

创建新虚拟环境

xxxxxxxxxx
conda create -n new_envs

查看新虚拟环境的路径

xxxxxxxxxx
conda env list

此时会发现环境路径已经修改，如果还没有修改，检查anaconda3文件夹是否有把灰色的锁图标，此时需要修改解锁相应的权限。

修改anaconda3文件夹权限，在anaconda3上级文件夹打开终端，输入：

xxxxxxxxxx
sudo chown -R username anaconda3
sudo chown -R username .conda

此时重新创建conda虚拟环境便会发现路径已经修改完成。
